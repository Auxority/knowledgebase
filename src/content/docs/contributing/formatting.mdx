## Extra Features
### YouTube
Include a youtube link on its own line to embed the video:
```mdx
https://www.youtube.com/watch?v=RzL_L29cGmQ
```
https://www.youtube.com/watch?v=RzL_L29cGmQ

### Lighterpack
Use the Lighterpack component to embed a gear list:
```js
//We're embedding https://lighterpack.com/r/sl6c8y

import Lighterpack from "@components/content/Lighterpack.astro";
<Lighterpack slug="sl6c8y" />
```
